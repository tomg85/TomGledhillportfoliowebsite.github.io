Data Cleaning with SQL

In my Data Cleaning project, I worked with a dataset containing information on global layoffs, which required extensive preprocessing and transformation using SQL. The dataset was messy, with inconsistencies in formatting, missing values, and duplicate entries that could distort any analysis.

I started by identifying and removing duplicate rows, ensuring that each entry represented a unique event. Then, I handled missing data by using SQL functions like `ISNULL` to either fill gaps with appropriate default values or exclude incomplete records. I also standardized the date and company name formats, converting them into a consistent format across the entire dataset. Through these steps, I was able to clean the data and prepare it for more in-depth analysis, ensuring its accuracy and reliability for subsequent reporting or insights.

Data Exploration with SQL

For my Exploratory Data Analysis (EDA) project in SQL, I worked with a dataset detailing layoffs around the world. The goal was to analyse trends, identify patterns, and uncover insights related to job cuts across various industries, regions, and time periods.

Having already cleaned and pre-processed the data (see my Data Cleaning project above), using SQL queries I explored the distribution of layoffs by company, sector, country, and year. I also conducted aggregations to identify key metrics, such as total layoffs per month using a rolling total. Ultimately, this project allowed me to develop a deeper understanding of SQL's data manipulation capabilities while also gaining insights into the global landscape of workforce reductions.

Data Visualisation with Tableau

For my Data Visualization project in Tableau, I analysed a dataset of AirBnB listings in Seattle, WA, to uncover key trends and insights related to pricing, location, and property characteristics. The dataset included variables such as price, room type, neighbourhood, availability, and number of reviews, which I used to create a series of interactive dashboards. These dashboards enabled users to explore how factors like proximity to downtown Seattle, property type, and seasonal trends influenced listing prices and booking patterns.

I incorporated various visualisation techniques, including maps, scatter plots, and bar charts, to help users easily identify pricing disparities across different neighbourhoods and understand the relationship between amenities and booking demand. Overall, the project aimed to provide a comprehensive view of the Seattle AirBnB market, enabling both hosts and potential guests to make data-driven decisions.

Web Scraping with Python

For my web scraping project, I utilized Python to extract product data from an Amazon webpage. By leveraging libraries such as requests, BeautifulSoup, and pandas, I was able to retrieve key product details, including titles, prices, ratings, and reviews. The project involved sending HTTP requests to the Amazon product page, parsing the HTML content to locate the relevant data, and then storing the extracted information in a structured format like a CSV file for further analysis. Special attention was given to handling pagination and respecting Amazon's terms of service by avoiding excessive requests and using delays between scrapes.

This project helped me gain hands-on experience in data extraction, web scraping techniques, and working with HTML parsing tools in Python.
